## AI-Powered RAG Document Assistant

An AI-powered Retrieval-Augmented Generation (RAG) chatbot built using n8n, Pinecone, OpenAI Embeddings, and OpenRouter.

## Workflow Architecture

 ! [Workflow Architecture](workflow-architecture.png)

## Features

- Automatic Google Drive file ingestion
- Document chunking and embeddings
- Pinecone vector database integration
- AI Agent for intelligent document Q&A
- Retrieval-Augmented Generation (RAG)

## Tech Stack

- n8n
- Pinecone
- OpenAI Embeddings
- OpenRouter
- Google Drive API

## Workflow

1. Detect new file upload in Google Drive
2. Download file automatically
3. Split document into chunks
4. Generate embeddings
5. Store vectors in Pinecone
6. Retrieve relevant chunks
7. Generate AI-powered answers

## Use Cases

1. AI Knowledge Base
2. PDF Chatbot
3.  Smart FAQ Assistant
4. Internal Company Assistant
